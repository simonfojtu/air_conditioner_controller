# Air conditioner controller

Controller for an air conditioner unit

manufacturer: ACOND
models: AS-12UR4SYDDA, AS-09UR4SYDDA


## IR receiver and display module

connected to main unit via 5 wires:
- YK (orange)
- KEY (yellow)
- SDA1 (greenish)
- +5V (blue)
- GND (purple)
